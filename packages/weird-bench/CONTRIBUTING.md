# Contributing

Thanks for helping move this prototype forward.

## Local setup

```bash
npm install
npm test
```

## Contribution guidelines

- Keep pull requests small and focused.
- Add or update tests for behavior changes.
- Update `README.md` or `ROADMAP.md` when user-facing behavior changes.
- For security-sensitive changes, especially package execution or encryption behavior, include a short threat-model note in the PR.
