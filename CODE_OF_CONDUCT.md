# Code of Conduct

This project uses a lightweight contributor covenant: be respectful, assume good intent, and keep collaboration focused on making the tools better.

## Expected behavior

- Be welcoming to new contributors and direct feedback at the work, not the person.
- Explain tradeoffs and decisions clearly.
- Respect boundaries around security research, disclosure timing, and maintainer capacity.
- Avoid harassment, discriminatory language, personal attacks, or sustained off-topic disruption.

## Reporting

If something in the project space makes collaboration unsafe or unproductive, contact the maintainers through a private channel or open a GitHub issue if privacy is not required. Maintainers may remove comments, close threads, or block participation to keep the project healthy.
